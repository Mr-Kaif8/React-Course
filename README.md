# ⚛️ React Course

A collection of React projects built while learning React.js. This repository contains hands-on projects that cover React fundamentals, hooks, routing, state management, and component-based development.

## 📚 Topics Covered

* JSX
* Components
* Props
* State (`useState`)
* Event Handling
* Conditional Rendering
* Lists & Keys
* Forms
* `useEffect`
* React Router
* Context API
* Custom Hooks
* API Fetching
* Tailwind CSS
* Project Deployment

## 🛠️ Tech Stack

* React.js
* JavaScript (ES6+)
* HTML5
* CSS3
* Tailwind CSS
* Vite

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/Mr-Kaif8/React-Course.git
```

Go to the project folder:

```bash
cd React-Course
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

## 🎯 Purpose

This repository is created to practice React concepts through real-world projects and improve frontend development skills.
