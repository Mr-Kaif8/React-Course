# React + Vite

 # 🧠 Quiz App
 Live Demo:http://kaif-quiz.vercel.app

An interactive and responsive Quiz Application built using **HTML, CSS, and JavaScript React**. The app allows users to answer multiple-choice questions, receive instant feedback, and view their final score.

## ✨ Features

* 📋 Multiple-choice questions
* ⚡ Instant answer validation
* 📊 Real-time score tracking
* 🔄 Next question navigation
* 📱 Responsive design
* 🚀 Fast and lightweight

## 🛠️ Technologies Used

* HTML5
* CSS3
* React
* JavaScript (ES6)

## 📂 Project Structure

```text
quiz-app/

```

## 🚀 Getting Started

1. Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/quiz-app.git
```

2. Open the project folder.

3. Run `index.html` in your browser.
